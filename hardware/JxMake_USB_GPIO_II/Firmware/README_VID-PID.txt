====================================================================================================
USB Vendor and Product IDs
====================================================================================================

The firmware source code in this directory uses the VID and PID from the original 'LUFA Dual CDC
Demo Application' (0x03EB and 0x204E).

While this is acceptable for private use or limited distribution, commercial or public reuse of
this VID and PID by others may be prohibited by USB-IF, even if the 'JxMake USB-GPIO Module II'
is technically still a 'LUFA Dual CDC' device.

Please edit 'ATxmega128A4U/Descriptors.c' and replace them with your own VID and PID as required.

====================================================================================================
