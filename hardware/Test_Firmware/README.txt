Test Firmware
====================================================================================================

----------------------------------------------------------------------------------------------------
All test firmware source code is not part of the JxMake program and is therefore not covered by
JxMake licenses.
----------------------------------------------------------------------------------------------------

Most firmware source code files do not include specific license text. All embedded SDKs are governed
by their respective licenses. Please refer to the subdirectories for further details.

----------------------------------------------------------------------------------------------------
JxMake utilizes the resulting HEX files in its test applications solely to validate the programmer
classes ('Prog*').
----------------------------------------------------------------------------------------------------

Some test firmware source code is also located at '../../test/src', as part of the build system test
suite.

Please note that some source code used to generate the HEX files may be missing, as it has been lost
over time.
